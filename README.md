# delicacy
美食分享类型网站,基于Node.js+Vue.js
## 后端:

**安装教程**

```shell
npm install
```

**运行项目**

```shell
nodemon index
```

## 前端:

**安装教程**

```shell
yarn install
```

**运行项目**

```shell
yarn serve
```

**打包项目**

```shell
yarn build
```
